# corrupted-file

## Challenge Text
* Can you find a way to fix our corrupted .jpg file?

## Hint
* No hints

## Solution
* Use a hex editor (https://hexed.it/) and append the missing bytes to the start of the file flag_mod.jpg
* FF D8 FF E0
* Save changes and open the file.
* Flag: `jctf{OaZdSdMo8F}`
